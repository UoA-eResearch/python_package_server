CeR Python Package Index

This is a CeR Python repository server that conforms to the Pep 503 specification, inspired by this blog post.

Pip will look for directories using the normalised name of your package ('ceradmin-common' in this case instead of 'ceradmin_common'). More details in the specification.

To install a package or a dependency from here:

Using pip:
pip install ceradmin_ingest --extra-index-url https://uoa-eresearch.github.io/python_package_server/

Using setup.py:
install_requires=['ceradmin_common'],
dependency_links=['git+https://github.com/UoA-eResearch/ceradmin_common#egg=ceradmin_common-0.0.1']

Note: Some repositories might be private. You'll be asked for your github credentials during the installation. The installation will fail if you are not authorized to access the repository.

More information about package versions will come soon.
